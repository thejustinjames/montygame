## 🚀 Setup Instructions

### 1. Open in Unity Hub

```bash
# Unity Hub should auto-detect this folder as a project
# If not, open Unity Hub → "Open" → select MontyGame-Unity folder
```

### 2. Reference MontyGame.Core

**Option A: Project Reference (Recommended)**
- In your IDE (Visual Studio/Rider), add project reference to `../src/MontyGame.Core/MontyGame.Core.csproj`
- Build to verify no errors

**Option B: Copy DLL**
```bash
# From repo root:
dotnet build src/MontyGame.Core
cp src/MontyGame.Core/bin/Debug/net8.0/MontyGame.Core.dll \
   MontyGame-Unity/Assets/Plugins/MontyGame.Core.dll
```

### 3. Create Test Scene

1. In Unity Editor: `Ctrl+N` (New Scene)
2. Save as: `Assets/Scenes/TestLevel_5Tiles.unity`
3. Set up:
   - 2D Camera
   - Canvas (for UI)
   - Physics2D settings

## 🚦 Troubleshooting

**Q: MontyGame.Core not found**
- A: Check project reference or copy DLL to Assets/Plugins/

**Q: Player jumps wrong (too high/low)**
- A: Adjust `jumpForce` in PlayerController.cs
- Example: jumpForce = 6.3 gives ~2.0 unit jump

**Q: Player feels sluggish**
- A: Increase `moveSpeed` in PlayerController.cs (recommended: 6)

**Q: Can't find TextMeshPro**
- A: First time using TextMeshPro? Unity prompts to import. Click "Import"
