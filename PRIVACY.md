# Privacy Policy

_Last updated: 2026-07-04_

This Privacy Policy explains how the **MontyGame** project handles information.
MontyGame is currently an early-stage game ideation project. This repository
holds concepts, planning materials, and research references — it does not run a
live service, application, or website, and it does not collect data from users.

## 1. Scope

This policy applies to the MontyGame repository and any related materials
maintained here. It does **not** cover GitHub itself; your interactions with
this repository (such as viewing, starring, forking, or opening issues) are
governed by [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

## 2. Information We Do Not Collect

The MontyGame project does not:

- Collect, store, or process personal data from visitors.
- Use cookies, analytics, tracking pixels, or third-party trackers.
- Require accounts, logins, or registration.
- Run servers or backends that receive user information.

Any information visible in this repository is content authored by the project
owner, not data gathered from you.

## 3. Third-Party Content and References

The `Research/` folder contains reference imagery gathered for inspiration and
mood-boarding. These materials are used for internal ideation only and are not
redistributed or offered for reuse. If you believe any referenced material
infringes your rights, please open an issue or contact the repository owner so
it can be reviewed and removed if appropriate.

## 4. Contributions and Public Interactions

If you interact with this repository (issues, pull requests, comments), any
information you submit becomes publicly visible on GitHub and is handled under
GitHub's own policies. Do not submit sensitive personal information.

## 5. Future Changes

If MontyGame grows into a playable game, app, or hosted service that collects
data, this policy will be updated to describe what is collected, why, and how
it is protected — before any such collection begins.

## 6. Contact

For privacy questions or requests, please open an issue in this repository or
contact the repository owner through GitHub.
