# MontyGame

A game ideation repository — a home for concepts, planning, research, and reference material as the game takes shape.

## Repository structure

| Folder | Purpose |
| --- | --- |
| `Docs/` | Design documents, specs, and written concepts. |
| `Planning/` | Roadmaps, task lists, and milestone planning. |
| `Research/` | Reference imagery, mood boards, and inspiration. |

## Status

Early ideation. Nothing is locked in yet — this repo captures ideas as they evolve.
